# Installation-Docker
Ce répertoire contient le fichier d'installation de docker engine pour toutes les versions de ubuntu.
Le fichier dockinst.sh s'inspire du fichier d'installation officiel disponible sur le site docker : https://docs.docker.com/engine/install/ubuntu/


# Docker-Installation
This repository contain the installation file of docker engine for each version of ubuntu.
The dockinst.sh file take source from the official installation file available at the docker website : https://docs.docker.com/engine/install/ubuntu/
